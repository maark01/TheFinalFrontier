1. kör terve:
-------------

- elkészíteni egy egyszerű frontend alkalmazást
- backend oldalon 24 óránként fetchelni a fenti API adatait és átadni egy pgsql adatbázis szervernek
- backend db-ből lekéri az adatokat és átadja frontendnek
- frontenden megjeleníteni
- dockerizálni db, backend és frontend részeket
- ha ez sikeres akkor tovább fejleszteni új modulokkal az alkalmazást

2. kör terve:
-------------

- haproxy használata és több docker konténer futtatása és round robint beállítani
- puppet mastert beállítani
- rendszer tesztelése, tönkre vágni az alkalmazást és figyelni, hogyan rakja rendbe a puppet

I.a - astronauts modul;
-----------------------

* /astronauts végpontról ezeket az adatokat kéri le:
    - id: number
    - name: string
    - status: string
    - agency: string
    - image: string
    - bio: string

    - a status, agency, image az API-ból egy-egy object a lekért astronaut objecten belül
    - bejárjuk egy map metódussal a lekért tömbünket, ezek után destruktálva egy forEach metódussal átadjuk a megfelelő astronautsMutation class metódusainak
    az adatokat, és az beilleszti majd a megfelelő pgsql táblákba

    * astronautsMutation metódusok:
    -------------------------------
        - assignAstronauts(id, name, bio) a paramétereket egy INSERT sql metódussal beilleszti a public.astronaut táblákba
        - addImage(id, name, imageUrl) a paramétereket egy INSERT sql metódussal beilleszti a public.image táblákba
        - assignStatus(statusId, statusName)  a paramétereket egy INSERT sql metódussal beilleszti a public.status táblába
        - assignAgenciesToAstronauts(agencyId, AtronautId) a paramétereket egy INSERT sql metódussal beilleszti a public.agencies_astronauts kapcsoló táblába
        - assignImageToAstronauts(imageId, AtronautId) a paramétereket egy INSERT sql metódussal beilleszti a public.astronauts_images kapcsoló táblába
        - assignStatusToAstronauts(statusId, astronautId)  a paramétereket egy INSERT sql metódussal beilleszti a public.astronauts_status kapcsoló táblába


    * pgsql tábla végpontok:
    ------------------------
        - astronauts(id: integer PK, name: character varying, bio: character varying)
        - status
